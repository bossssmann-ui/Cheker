# Cheker

**Инструмент для проверки сайтов на SEO-ошибки и технические проблемы.**

Cheker анализирует любую веб-страницу и формирует подробный отчёт с оценкой по 100-балльной шкале.

## Что проверяется

| Проверка | Описание |
|---|---|
| HTTP-статус | Доступность страницы |
| HTTPS | Использование защищённого протокола |
| Время загрузки | Рекомендуемый порог — до 2 с |
| `<title>` | Наличие и длина (10–70 символов) |
| Meta description | Наличие и длина (50–160 символов) |
| `<h1>` | Наличие и единственность |
| Атрибут `alt` у изображений | Все изображения должны иметь alt |
| Viewport | Адаптивность для мобильных устройств |
| Canonical | Канонический URL |
| Open Graph | og:title, og:description, og:image |
| robots.txt | Наличие файла на сервере |
| sitemap.xml | Наличие карты сайта |

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python cli.py https://example.com
```

Или без схемы (будет добавлен `https://` автоматически):

```bash
python cli.py example.com
```

### Параметры

| Флаг | Описание |

### Использование как библиотека

```python
from checker import analyze

report = analyze("https://example.com")
print(f"Оценка: {report.score}/100")
for check in report.checks:
    status = "✓" if check.passed else "✗"
    print(f"  {status} {check.message}")
```
