# Cheker

**Инструмент для проверки сайтов на SEO-ошибки и технические проблемы.**

Cheker анализирует любую веб-страницу и формирует подробный отчёт с оценкой по 100-балльной шкале.

> 🌐 **[Страница проекта](https://bossssmann-ui.github.io/Cheker/)** — подробное описание и онлайн-запуск проверки.

## Проверить сайт без программирования (через GitHub)

> Не умеете программировать? Не страшно — проверка запускается прямо в браузере.

1. Откройте вкладку **Actions** в этом репозитории.
2. В левой панели выберите **«Проверка сайта»**.
3. Нажмите кнопку **«Run workflow»** (зелёная кнопка справа).
4. Введите URL сайта (например, `pacificstar.ru`) и нажмите **«Run workflow»**.
5. Дождитесь завершения задания (обычно 1–2 минуты).
6. Откройте завершённое задание — в разделе **«Summary»** отобразится полный отчёт.
7. Файл `report.txt` также доступен как **артефакт** для скачивания (хранится 30 дней).

Кроме того, сайт **pacificstar.ru** проверяется автоматически **каждый день в 06:00 UTC**.

## Что проверяется

| Проверка | Описание |
|---|---|
| HTTP-статус | Доступность страницы |
| HTTPS | Использование защищённого протокола |
| Время загрузки | Рекомендуемый порог — до 2 с |
| `<title>` | Наличие и длина (10–70 символов) |
| Meta description | Наличие и длина (50–160 символов) |
| `<h1>` | Наличие и единственность |
| Атрибут `alt` у изображений | Все изображения должны иметь alt |
| Viewport | Адаптивность для мобильных устройств |
| Canonical | Канонический URL |
| Open Graph | og:title, og:description, og:image |
| robots.txt | Наличие файла на сервере |
| sitemap.xml | Наличие карты сайта |
| Lang | Атрибут `lang` у тега `<html>` |
| Favicon | Наличие иконки сайта |
| Structured Data | JSON-LD (Schema.org) блоки |
| Twitter Card | twitter:card, twitter:title, twitter:description |
| Heading structure | Корректная иерархия заголовков (h1–h6) |

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python cli.py https://example.com
```

Или без схемы (будет добавлен `https://` автоматически):

```bash
python cli.py example.com
```

### Использование как библиотека

```python
from checker import analyze

report = analyze("https://example.com")
print(f"Оценка: {report.score}/100")
for check in report.checks:
    status = "✓" if check.passed else "✗"
    print(f"  {status} {check.message}")
```
