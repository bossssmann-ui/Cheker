Что проверяет Cheker

Cheker анализирует HTML-страницу и выдаёт структурированный отчёт с пометками «пройдено / предупреждение / ошибка» по каждому параметру.

🌐

HTTP-статус

Проверяет доступность страницы и код ответа сервера.

🔒

HTTPS

Использование защищённого протокола — обязательное требование Google.

⚡

Время загрузки

Рекомендуемый порог — до 2 с. Медленный сайт теряет позиции.

📝

Тег <title>

Наличие и длина заголовка страницы (10–70 символов).

📄

Meta description

Наличие и длина описания страницы (50–160 символов).

🏷️

Тег <h1>

На каждой странице должен быть ровно один тег H1.

🖼️

Alt у изображений

Все изображения должны иметь текстовое описание (alt).

📱

Viewport

Наличие мета-тега для адаптивности на мобильных устройствах.

🔗

Canonical

Канонический URL предотвращает дублирование контента.

📊

Open Graph

Теги og:title, og:description, og:image для социальных сетей.

🤖

robots.txt

Файл инструкций для поисковых роботов на сервере.

🗺️

sitemap.xml

Карта сайта помогает поисковым системам обходить страницы.

🌍

Lang

Атрибут lang у тега <html> важен для SEO и доступности.

🎨

Favicon

Иконка сайта через тег <link> или стандартный /favicon.ico.

🧩

Structured Data

JSON-LD разметка (Schema.org) для расширенных сниппетов.

🐦

Twitter Card

Мета-теги twitter:card, twitter:title, twitter:description.

📐

Структура заголовков

Корректная иерархия h1–h6 без пропущенных уровней.

Проверка сайта без программирования

Запустить проверку можно прямо в браузере — без установки Python и без командной строки.

  1. Откройте вкладку Actions в этом репозитории.
  2. В левой панели выберите «Проверка сайта».
  3. Нажмите «Run workflow» (зелёная кнопка справа).
  4. Введите URL сайта, например pacificstar.ru, и нажмите «Run workflow».
  5. Дождитесь завершения задания (обычно 1–2 минуты).
  6. Откройте завершённое задание — в разделе «Summary» отобразится полный отчёт. Файл report.txt доступен для скачивания 30 дней.

Установка и локальный запуск

1. Установить зависимости

pip install -r requirements.txt

2. Запустить проверку

python cli.py https://example.com

Можно указать URL без схемы — https:// будет добавлен автоматически:

python cli.py example.com

Параметры

  • --no-color — отключить цветной вывод
  • --repeat / -r — повторить последнюю проверку

Использование как библиотека

from checker import analyze

report = analyze("https://example.com")
print(f"Оценка: {report.score}/100")
for check in report.checks:
    status = "✓" if check.passed else "✗"
    print(f"  {status} {check.message}")
